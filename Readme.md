## Setup Steps

Clone the GitHub URL

### Backend Setup

```bash
npm install
npm run dev
```

Can run test using:

```bash
npm test
```

### Frontend Setup

```bash
npm install
npm run dev
```

## TechStack used

Reactjs, Expressjs, Nodejs
